## Overview

ZonaSegura CDMX is an interactive geospatial web application that visualizes crime data in Mexico City, combining official crime records with real-time citizen reports.

## Key Features

- Built an interactive geospatial web application to visualize crime data in Mexico City
- Processed 808k+ crime records, optimizing 200MB+ CSVs into 30MB GeoJSON datasets
- Implemented heatmaps, 1km² risk zones, and dynamic filters by type, date, and location
- Added a citizen reporting system with map markers

## Technologies

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React-Leaflet
- Turf.js

## Project Period

June 2025 – October 2025
